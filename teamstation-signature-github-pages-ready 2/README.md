# TeamStation AI Email Signature Studio

This repository hosts the TeamStation AI email signature page on GitHub Pages.

## Use The Signature Page

Open:

```text
https://teamstationaiaxiomvertex.github.io/signature/
```

Then:

1. Choose a preset or enter your details.
2. Click **Copy signature**.
3. Paste into your email app's signature settings.
4. Send yourself a test email.

Use **Copy HTML** only if you are reviewing or editing source HTML.

## Where To Paste It

Gmail:

```text
Settings -> See all settings -> General -> Signature
```

Outlook:

```text
Settings -> Accounts -> Signatures
```

Apple Mail:

```text
Mail -> Settings -> Signatures
```

## If The Logo Does Not Show

1. Refresh the signature page.
2. Click **Copy signature** again.
3. Paste it into the email app again.
4. Send another test email.

The logo URL used by installed signatures is:

```text
https://teamstationaiaxiomvertex.github.io/signature/assets/teamstation-logo-160.png
```

That URL must stay public.
