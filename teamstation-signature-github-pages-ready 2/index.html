Install workflow

Use the deployed GitHub Pages URL, not the raw GitHub file view.

  1. Open the signature studio. Use the public GitHub Pages page so the logo URL is reachable by email clients.
  2. Choose or enter the team member. Confirm name, title, email, phone, calendar, and optional links.
  3. Copy the rendered signature. Paste into Gmail Settings, Outlook Signatures, or Apple Mail Signatures.
  4. Send a test email. Verify logo rendering, links, phone numbers, and legal footer before using externally.
